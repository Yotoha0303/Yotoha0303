1、
```
很多人无法意识到“自己做出的某些行为可能会对他人造成伤害”，这是一种“无意之恶”。
不要因为他人的错误而感到悲伤和痛苦，请你在挣扎和悲伤中，相信“他人对你的伤害是一种故意的行为”，
感受自己面对的“人性之恶”，感受自己的无力、挫败和痛苦。从中吸取经验，让自己脱离内心的沼泽。
理解“人性”，并使得自己成长才是最重要的。也请你放下对于他们的“恶意”的理解和放弃自我的想法，
在下一次不让他们对自己造成影响，就是你这次所收获的成长。
```

2、
```
有些人可能发掘到，我的行为有时会变得很“奇怪”，但是他们不知道为什么。
我知道我的行为突然改变是为什么，那不仅仅是因为我敏感，而是我知道他们做出的某些事情的背后是“恶意”，
于是我就开始退缩、逃避。在他人眼中看来，我是玻璃心、脆弱的。
我也承认，但这是由于我知道真正的事实是怎么样的，而不是由于我脆弱。
```

3、
```
我很多次表达出我的真实想法，偶尔我也意识到自己表达出的内容会被长辈反驳、打断的时候意识到，
我的表达有些存在“欺骗”的意味，在我每当意识到这个问题时，我都会问自己“如果对方不喜欢我说这样的话，
那么我是否也会让自己主动的去说欺骗他人的话吗？”。但在这次，我想“既然我无法表达出我真实的想法，
那么我就用实力让他们闭嘴，而不是表达”。我，并不想要让自己去“有无意识”的讲假话，来欺骗他人。
```

5、
```
尼采曾经说过“其实人跟树是一样的，越是向往高处的阳光，它的根就越要伸向黑暗的地底”。
这句话从我的童年开始，深深的扎根在我的内心。我一路走来，我承受了很多误解、委屈、失望，我扎根于泥土，却向往天上的阳光。
```

7、
```
在大学时，我曾经意识到“我身边的人缺乏驱使'自己'前进的动力，他们的人生前途可能在这一刻被毁灭”，我很失望，但是我选择了这里。
我，需要去面对。于是，我在内心中构建出了一个“强大而完美的自己”。他的存在很模糊，但是很真实。
他在每一次接受我在内心中呼喊“我要变强”时出现。它，是我的英雄，也是我的噩梦。我有一个永远可以追随的榜样，
但是我永远也无法超过他。我开始自卑，开始认为“强大”并非来自于外界的认同，于是我开始逐渐逃避现实，我对任何需要自己去完成自己的事情面前
变得畏畏缩缩。甚至，在他人发自真心的评价我的努力时，我仅仅认为这只是“奉承”，我并没有理会。
我辜负了我的英雄，我无法直面他的强大。我辜负了自己，我无法面对需要自己去做的事情。我辜负了真心认可我的他人，我把他们的认可当作了“奉承”。
```
